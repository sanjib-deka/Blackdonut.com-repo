BlackDonut — Project Overview 

BlackDonut is an authentic food discovery platform that helps people explore real local dishes, discover hidden gems, and experience the stories behind iconic foods. Unlike traditional food delivery apps, BlackDonut focuses on experience, authenticity, and local exploration rather than just ordering.

Technical Structure
The project is divided into two repositories:

Frontend — User-facing application with UI, reels, and search features.

Backend — Server-side code with APIs, database management, and content handling.

For more technical details, setup instructions, and contribution guidelines, please visit the respective repositories:

Backend Repo: [[link](https://github.com/sanjib-deka/Blackdonut.com-repo/tree/main/backend%20blackdonut-git)]

Vision
To redefine food exploration, connecting users with authentic culinary experiences, cultural stories, and local communities.
